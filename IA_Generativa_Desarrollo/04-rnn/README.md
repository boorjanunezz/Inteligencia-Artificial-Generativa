# 🔁 RNN y LSTM

Este bloque introduce las redes neuronales recurrentes y su evolución práctica con LSTM. Aquí encontrarás la guía teórica y la práctica aplicada a series temporales para trabajar predicción secuencial.

## 📖 Contenido Disponible

### 📘 Teoría

Guía didáctica centrada en RNN y LSTM: secuencias, estados ocultos, problemas de memoria, puertas de la LSTM, implementación en Keras y criterios para elegir la arquitectura adecuada.

➡️ **[Ver guía teórica](./TEORIA.md)**

**Temas cubiertos:**
- Qué es una RNN y cómo procesa secuencias
- Problemas de las RNN clásicas
- Cómo funciona una LSTM
- Ejemplos de implementación en Keras
- Cuándo usar RNN o LSTM

### 🎯 Práctica

Caso de estudio aplicado a la predicción del precio del oro con LSTM, usando series temporales y variables macroeconómicas.

➡️ **[Ver ejercicio práctico](./PRACTICA.md)**

**Contenido incluido:**
- Preparación de datos temporales
- Construcción de secuencias con ventana histórica
- Diseño de una LSTM para regresión
- Entrenamiento y evaluación temporal
- Análisis de resultados y limitaciones

## 🚀 Cómo usar este contenido

1. **Comienza con la teoría** - Revisa RNN y LSTM para entender la lógica secuencial.
2. **Practica lo aprendido** - Aplica el caso del oro para trabajar una serie temporal real.
3. **Consulta cuando lo necesites** - Usa la teoría como apoyo mientras ajustas la ventana, la arquitectura y las métricas.

## 💡 Recomendaciones

- **Empieza por la RNN simple** para fijar la intuición antes de pasar a LSTM.
- **Cuida la forma de las secuencias**: el orden temporal es parte del dato.
- **Escala las variables** antes de entrenar.
- **Usa validación temporal** para no romper el orden cronológico.
